# Bloch Sphere Visualizer

An interactive web-based Bloch Sphere Visualizer that demonstrates qubit state representation in quantum computing using Three.js.

---

## 🚀 Overview

This project visualizes a qubit on a 3D Bloch Sphere. Users can adjust the angles **θ (theta)** and **φ (phi)** to explore different quantum states in real time. The application dynamically updates the state vector, bra-ket notation, and probability amplitudes.

---

## 🎯 Features

- 🌐 3D Bloch Sphere visualization using Three.js  
- 🎚️ Adjustable θ (theta) and φ (phi) sliders  
- 🔴 Real-time qubit state vector (arrow representation)  
- 🧮 Bra-ket notation display: |ψ⟩ = a|0⟩ + b|1⟩  
- 📊 Probability calculation with visual bars (P(0), P(1))  
- 🎲 Measurement simulation (quantum collapse)  
- 🏷️ State labels |0⟩ and |1⟩  
- 🎨 Clean and interactive UI

## 🎮 How to Use

- Move the **θ (theta)** slider to change vertical angle  
- Move the **φ (phi)** slider to rotate horizontally  
- Observe the **state vector (arrow)** change  
- Check updated **probabilities and state equation**  
- Click **Measure** to simulate quantum measurement  

---

## 📸 Output

- A rotating 3D Bloch Sphere  
- Dynamic arrow showing qubit state  
- Real-time probability visualization  

---

## 📌 Formula Used

The qubit state is represented as:

|ψ⟩ = cos(θ/2)|0⟩ + e^(iφ) sin(θ/2)|1⟩
